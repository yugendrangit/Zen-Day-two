DOCUMENT OBJECT : 

1. The document object represents the structure of the web page as a whole and provides access to the content of the page
as well as methods for manipulating that content. It contains information about the content of the page.

2. For Example : the URL, the title, the content, and the links.

3. The document object is a property of the window object and is also a global object in JavaScript, which means that it is available in all contexts throughout your code.
   You can access it directly using the document keyword.

4. Syntax : document.propertyName

5. JavaScript Document methods are: getElementById(), createElement(), querySelector(), querySelectorAll(), etc.

6. Document object is available globally and it is an object of the browser.
